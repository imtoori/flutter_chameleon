# flutter_chameleon

Flutter package to show platform **dependent** widgets.

## Usage

Just use chameleon widgets instead of material or cupertino and they will show up depending on the hosting platform.

## Widget included

* activity indicator
* alert dialog
* bottom navigationbar
* button
* dialog
* slider
* switch

More to come..
